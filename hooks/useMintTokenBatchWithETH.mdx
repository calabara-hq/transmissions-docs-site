## Returns
### mintTokenBatchWithETH
Function to mint a batch of tokens with ETH. See [mintTokenBatchWithETH](/sdk/uplink/tokens/mintTokenBatchWithETH) args.

- **type**: `({
    channelAddress: string;
    to: string;
    tokenIds: bigint[];
    amounts: number[];
    mintReferral: string;
    data: string;
    transactionOverrides?: TransactionOverrides;
}) => Promise<Log[] | undefined>`

### status
- **type**: `ContractExecutionStatus`
```typescript
type ContractExecutionStatus =
  | 'pendingApproval'
  | 'txInProgress'
  | 'complete'
  | 'error'
```
the status of the transaction

### txHash
- **type**: `string`

the transaction hash of the transaction

### error
- **type**: `any`

the revert message if the transaction fails
