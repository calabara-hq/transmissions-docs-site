## Usage
<CodeGroup>
```typescript Component.tsx

const MyComponent = () => {
    const { 
        mintTokenBatchWithERC20,
```

## Returns
### mintTokenBatchWithERC20
Function to create a new token. See [mintTokenBatchWithERC20](/sdk/uplink/tokens/mintTokenBatchWithERC20) args.

- **type**: `({
    channelAddress: string;
    to: string;
    tokenIds: bigint[];
    amounts: number[];
    mintReferral: string;
    data: string;
    transactionOverrides?: TransactionOverrides;
}) => Promise<Log[] | undefined>`

### status
- **type**: `ContractExecutionStatus`
```typescript
type ContractExecutionStatus =
  | 'pendingApproval'
  | 'txInProgress'
  | 'complete'
  | 'error'
```
the status of the transaction

### txHash
- **type**: `string`

the transaction hash of the transaction

### error
- **type**: `any`

the revert message if the transaction fails
